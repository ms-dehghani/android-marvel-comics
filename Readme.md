
# Android Marvel Comics App
* Our goal is to implement a movie app using Marvel API and modern Android technologies

## Architecture
* Clean Architecture

### Technologies

Project is created with:
* MVVM
* Jetpack Compose
* Jetpack Compose Navigation (Single activity, zero fragments)
* Paging 3
* Caching
* Hilt
* Retrofit
* kotlin Coroutine
* Room
* Clean Architecture
* Modular structure
* Unit tests
* Ui tests
* CI/CD (optional)
